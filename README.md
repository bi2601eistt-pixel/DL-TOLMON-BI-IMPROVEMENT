# DL-TOLMON-BI-IMPROVEMENT

Deep Learning Based Toll Gate Traffic Prediction and Decision Support System for Efficient Urban Transportation.
Proyek ini berisi notebook analisis, dataset, dan dashboard HTML untuk monitoring kondisi gerbang tol.

## Isi Repository

- `notebooks/` berisi notebook Jupyter untuk analisis dan pengembangan model.
- `data/` berisi dataset data_tol.xlsx dan data_tol_clean_rebuilt.csv.
- `dashboard/` berisi dashboard HTML interaktif dashboard_live.html dan patched version dashboard_live_patched.html
- `README.md` berisi dokumentasi proyek.

## Cara Membuka

1. Buka notebook di folder `notebooks/`.
2. Dataset tersedia di folder `data/`.
3. Dashboard dapat dibuka dari file `dashboard/dashboard_live_patched.html`.

## Catatan

Dashboard menggunakan data yang sudah dimasukkan langsung ke dalam file HTML.
